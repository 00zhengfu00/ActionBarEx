# ActionBarEx
## 高拓展高自定义性ActionBar，完美替代Android系统默认

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.goweii:ActionBarEx:v2.1.0'
	}
